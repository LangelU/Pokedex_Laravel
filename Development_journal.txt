	- Número
	- Nombre
	- Imágen correspondiente
	- Tipo primario
	- Tipo secundario (Si tiene)
	- Título
	- Altura en metros
	- Peso en Kilogramos
	- Descripción
	- Género
	- Tasa de captura
	- Grupo huevo
	- Habilidades
	- Estadísticas
		- Salud
		- Ataque
		- Defensa
		- Ataque especial
		- Defensa especial
		- Velocidad
		- Total
	- Pre-evolución si la tiene
	- Evolución si la tiene
	- Mega evolución si la tiene
1.3) Para las estadísticas se implementarán barras para darle una visualización más gráfica
1.4) Las pre-evoluciones, megas, y evoluciones se mostrarán con imágenes
1.5) El color del banner, así como del recuadro que encierre la imágen, deberán cambiar de color en función del tipo principal del pokémon mostrado
1.6) La vista principal tendrá una barra para búsqueda por número
1.7) La vista principal tendrá una barra para búsqueda por nombre
1.8) La vista principal contendrá un filtro para cada tipo
1.9) Solo se permitirá un máximo de 2 filtros activos simultáneamente
1.10) Si existen 2 filtros activos, los demás se deshabilitarán hasta que se libere uno
1.11) Filtrar de mayor número a menor
1.12) Filtrar de menor número a mayor
